# Room Rent

Find and list short-term rentals on [ProAppStore](https://proappstore.online).

Browse listings, book stays, host your own place — with photos, reviews, and real-time booking management.

Live: <https://room-rent.proappstore.online>

- Dev: `pnpm install && pnpm dev`
- Build: `pnpm build`
- Deploy: `git push origin main`
